# Vietnamese Sentiment Analysis

## Giới thiệu

Đây là ứng dụng phân tích cảm xúc văn bản tiếng Việt. Hệ thống có thể dự đoán bình luận là Positive hoặc Negative bằng mô hình Machine Learning.

## Công nghệ sử dụng

- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression
- Joblib
- python-docx

## Chức năng

- Phân tích một bình luận
- Phân tích đoạn dài
- Phân tích file TXT/DOCX
- Phân tích CSV hàng loạt
- Xem dataset
- Xem đánh giá mô hình

## Cách chạy project

Cài thư viện:

```bash
pip install -r requirements.txt
```

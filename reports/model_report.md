# Vietnamese Sentiment Analysis - Model Report

## 1. Giới thiệu dự án

Dự án này xây dựng ứng dụng phân tích cảm xúc văn bản tiếng Việt. Người dùng có thể nhập bình luận, đoạn văn dài hoặc tải file TXT/DOCX/CSV để hệ thống dự đoán cảm xúc là Positive hoặc Negative.

## 2. Mục tiêu

- Xây dựng hệ thống phân loại cảm xúc văn bản tiếng Việt.
- Áp dụng quy trình Machine Learning cơ bản cho bài toán NLP.
- Tạo giao diện web trực quan bằng Streamlit.
- Cho phép phân tích một câu, đoạn dài, file Word/TXT và CSV.

## 3. Công nghệ sử dụng

- Python
- Streamlit
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression
- Joblib
- python-docx

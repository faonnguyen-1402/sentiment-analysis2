# Experiment Notes

## Experiment 1: Initial Dataset

Ban đầu dataset còn ít nên accuracy chưa cao. Khi test size nhỏ, chỉ cần model dự đoán sai vài mẫu thì accuracy giảm mạnh.

Ví dụ:

```txt
Test size = 4
Sai 3 mẫu
Accuracy = 1/4 = 0.25
```
